# Screenshots

Ảnh minh họa trong README. Chụp lại khi UI thay đổi:

1. `docker compose up` (hoặc backend + `cd frontend && npm run dev`)
2. Mở http://localhost:5173
3. Lưu PNG vào thư mục này (đúng tên file):

| File | Trang |
|------|-------|
| `dashboard.png` | `/` — Tổng quan |
| `sessions.png` | `/sessions` — Tài khoản |
| `roster.png` | `/roster` — Sổ tài khoản |
| `dialogs.png` | `/dialogs` — Tin nhắn |
| `groups.png` | `/groups` — Nhóm & kênh |
| `tasks.png` | `/tasks` — Tác vụ hàng loạt |
| `conversation.png` | `/conversation` — Hội thoại tự nhiên |
| `audit.png` | `/audit` — Nhật ký hoạt động |
| `security.png` | `/security` — Bảo mật |
| `health.png` | `/health` — Trạng thái API |

Gợi ý: light theme, cửa sổ ~1280px, có dữ liệu mẫu (session + audit) để README trông đầy đủ.
